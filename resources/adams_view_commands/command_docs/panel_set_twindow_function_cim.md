# panel set twindow_function cim

Configures the panel test window function to return a contact impact measure (CIM) value.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `contact_name` | Object | Name of the contact to query. |
| `incident` | Integer | Incident identifier for the contact query. |
| `j_flag` | Boolean | Whether to return the value on the J geometry instead of the I geometry. |
