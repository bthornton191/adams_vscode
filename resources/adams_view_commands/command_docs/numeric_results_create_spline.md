# numeric_results create spline

Allows you to create a result set from spline data points.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `spline_name` | An Existing Spline | Specifies an existing spline. |
| `new_result_set_name` | A New Result Set | Allows you to identify the new result set name to be created with this operation. |
| `z` | Real | Real |
| `units` | Units_Type_With_Calc | Allows you to specify the type of units to be used for the new result set component. |
