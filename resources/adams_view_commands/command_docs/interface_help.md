# interface help

Allows you to display help content for a specified GUI element or topic.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `gui_name` | String | Specifies the name of the GUI element for which help is displayed. |
| `file_name` | String | Specifies the help file to open. |
| `link_name` | String | Specifies the anchor or topic link within the help file to navigate to. |
