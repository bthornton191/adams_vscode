# xy_plots template pan

Pans the view of an XY plot template along one or both axes.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `plot_name` | Object | Name of the XY plot to pan. |
| `pan_type` | String | Axis to pan (`x`, `y`, or `xy`). |
| `pan_xstart` | Real | Starting X value of the panned view. |
| `pan_xend` | Real | Ending X value of the panned view. |
| `pan_ystart` | Real | Starting Y value of the panned view. |
| `pan_yend` | Real | Ending Y value of the panned view. |
