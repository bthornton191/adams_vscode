# runtime_function create

Allows you to create a runtime function, which is a user-defined expression function that can be evaluated during simulation.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `runtime_function_name` | A New Runtime Function | Specifies the name of the new runtime function. |
| `text_of_expression` | String | Specifies the Adams function expression that defines the runtime function. |
| `argument_names` | String | Specifies the names of the arguments accepted by the runtime function. |
