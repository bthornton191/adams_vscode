# defaults stereo

The defaults stereo command is used to set the default stereo 3D viewing parameters for the Adams View graphics window.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `depth_of_field` | Real | Specifies the default depth-of-field for stereo rendering. |
| `eye_separation` | Length | Specifies the default distance between the virtual left and right eye positions. |
| `parallax` | Real | Specifies the default screen parallax used to control the apparent depth of the stereo image. |
| `eye_position` | Real | Specifies the default eye position used for stereo projection. |
