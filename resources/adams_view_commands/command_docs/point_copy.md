# point copy

Allows you to create a copy of an existing point. The copy is identical to the original with the exception of the point name.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `point_name` | An Existing Point | Specifies the point to be modified |
| `new_point_name` | A New Point | Specifies the name of the new point to be created, which will be a copy of an existing point |
