# list_info geometry

Displays information about geometry. The information can be displayed on the terminal, in a file, or both.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `geometry_name` | Existing geometry | Specify an existing geometry name. |
| `arc_name` | Existing arc | Specify an existing arc name. |
| `block_name` | Existing block | Specify an existing block name. |
| `circle_name` | Existing circle | Specify an existing circle name. |
| `cylinder_name` | Existing cylinder | Specify an existing cylinder name. |
| `frustum_name` | Existing frustum | Specify an existing frustum name. |
| `force_name` | Existing force | Specify an existing force name. |
| `outline_name` | Existing outline | Specify an existing outline name. |
| `polyline_name` | Existing polyline | Specify an existing polyline name. |
| `spring_damper_name` | Existing spring damper | Specify an existing spring damper name. |
| `gstring_name` | Existing gstring | Specify an existing gstring name. |
| `ellipsoid_name` | Existing ellipsoid | Specify an existing ellipsoid name. |
| `torus_name` | Existing torus | Specify an existing torus name. |
| `revolution_name` | Existing revolution | Specify an existing revolution name. |
| `extrusion_name` | Existing extrusion | Specify an existing extrusion name. |
| `plane_name` | Existing plane | Specify an existing plane name. |
| `brief` | On/Off | Specifies whether to use the brief form (default) or an extended form of the requested information to be displayed. |
| `write_to_terminal` | On/Off | Specify if the information requested is to be sent to the informational window or not. |
| `file_name` | String | Specify that the information requested is to be sent to a file with the name specified with the parameter. |
