# defaults name_generation part

Allows you to set the name-generation prefixes for part entities.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `part_prefix` | String | Specifies the text string to be used as a prefix when Adams View generates names for a part. |
| `diff_prefix` | String | Specifies the text string to be used as a prefix when Adams View generates names for a differential equation. |
| `transfer_function_prefix` | String | Specifies the text string to be used as a prefix when Adams View generates names for a transfer function. |
| `linear_state_equation_prefix` | String | Specifies the text string to be used as a prefix when Adams View generates names for a linear state equation. |
| `general_state_equation_prefix` | String | Specifies the text string to be used as a prefix when Adams View generates names for a general state equation. |
