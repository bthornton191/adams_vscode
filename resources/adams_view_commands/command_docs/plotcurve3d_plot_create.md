# plotcurve3d plot create

Creates a new empty 3D curve plot in the Adams View workspace.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `plot_name` | String | Name for the new 3D plot. |
| `create_page` | Boolean | Whether to create a new page for the plot. |
| `display_page` | Boolean | Whether to display the page after creation. |
| `page_name` | String | Name of the page on which to place the plot. |
| `view_name` | String | Name of the view in which to display the plot. |
| `bounding_box` | Array | Position and size of the plot bounding box. |
| `graph_volume` | Array | 3D volume definition for the graph axes. |
| `grid_lines` | Boolean | Whether to display grid lines. |
| `grid_color` | String | Color of the grid lines. |
| `grid_style` | String | Style of the grid lines. |
| `grid_thickness` | Integer | Thickness of the grid lines. |
| `title` | String | Main title text for the plot. |
| `subtitle` | String | Subtitle text for the plot. |
