# xy_plots template replace_simulation

Allows you to update curves from one analysis on a plot with new data from another analysis.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `plot_name` | Existing Plot | Specifies name of an existing plot. |
| `old_run_name` | Existing Analysis | Specifies name of analysis which is to be replaced. |
| `new_run_name` | Existing Analysis | Specifies the name of the simulation containing the new simulation results. |
