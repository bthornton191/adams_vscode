# plotcurve3d curve create

Creates a new curve on an existing 3D plot from explicit coordinate data.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `curve_name` | String | Name for the new 3D curve. |
| `x_values` | Array | Array of X coordinate values. |
| `y_values` | Array | Array of Y coordinate values. |
| `z_values` | Array | Array of Z coordinate values. |
| `x_units` | String | Units for the X coordinate values. |
| `y_units` | String | Units for the Y coordinate values. |
| `z_units` | String | Units for the Z coordinate values. |
| `legend_text` | String | Legend label for the curve. |
| `auto_color` | Boolean | Whether to automatically assign a color to the curve. |
| `color` | String | Explicit color for the curve. |
| `line_type` | String | Line style (e.g., `solid`, `dashed`). |
| `thickness` | Integer | Line thickness. |
| `run_name` | String | Name of the simulation run associated with the curve. |
