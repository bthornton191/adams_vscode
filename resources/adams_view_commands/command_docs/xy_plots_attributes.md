# xy_plots attributes

Sets visual attributes for an existing XY plot, including grid, legend, axis, and color settings.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `plot_name` | String | Name of the XY plot to modify. |
| `grid_lines` | Boolean | Whether to display grid lines. |
| `tic_marks` | Boolean | Whether to display tick marks on axes. |
| `legend` | Boolean | Whether to display the curve legend. |
| `axis_visibility` | Boolean | Whether to display the plot axes. |
| `axis_labels` | Boolean | Whether to display axis labels. |
| `symbol` | String | Symbol style for data points. |
| `all_color` | String | Global color override for all plot elements. |
| `title_color` | String | Color of the plot title text. |
| `subtitle_color` | String | Color of the plot subtitle text. |
| `haxis_label_color` | String | Color of the horizontal axis label. |
| `vaxis_label_color` | String | Color of the vertical axis label. |
| `axis_numbers_color` | String | Color of the axis tick number labels. |
| `grid_color` | String | Color of the primary grid lines. |
| `grid_style` | String | Line style for the primary grid lines. |
| `grid_thickness` | Integer | Thickness of the primary grid lines. |
| `secondary_grid_color` | String | Color of the secondary grid lines. |
| `secondary_grid_style` | String | Line style for the secondary grid lines. |
| `secondary_grid_thickness` | Integer | Thickness of the secondary grid lines. |
| `tic_color` | String | Color of the tick marks. |
| `border_color` | String | Color of the plot border. |
| `border_style` | String | Line style of the plot border. |
| `border_thickness` | Integer | Thickness of the plot border. |
| `axis_color` | String | Color of the axis lines. |
| `scientific_notation` | Boolean | Whether to use scientific notation for axis numbers. |
| `trailing_zeros` | Boolean | Whether to display trailing zeros on axis numbers. |
| `decimal_places` | Integer | Number of decimal places for axis number labels. |
| `legend_placement` | String | Location of the legend on the plot. |
| `simulation_legend_placement` | String | Location of the simulation legend on the plot. |
| `zero_line` | Boolean | Whether to display a line at zero. |
| `graph_area` | Array | Fractional position and size of the graph area within the plot. |
| `zoom_axis` | String | Which axis to zoom (`x`, `y`, or `both`). |
