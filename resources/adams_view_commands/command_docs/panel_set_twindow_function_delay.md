# panel set twindow_function delay

Configures the panel test window function to use a delayed expression.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `delayed_expr` | String | The expression whose delayed value is returned. |
| `delay_mag` | Real | The time delay magnitude. |
| `initial_expr_val` | Real | Value to return before the delay time has elapsed. |
| `delay_logic_array` | Array | Array used internally for delay state storage. |
