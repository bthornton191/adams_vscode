# interface tree_navigator

Allows you to configure the Adams View tree navigator panel.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `embed_container_name` | String | Specifies the name of the container in which the tree navigator is embedded. |
| `updates_enabled` | Boolean | Specifies whether the tree navigator updates automatically when the model changes. |
| `refresh` | Boolean | Specifies that the tree navigator should refresh its content immediately. |
