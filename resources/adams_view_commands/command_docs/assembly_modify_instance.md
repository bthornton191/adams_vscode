# assembly modify instance

Allows you to modify an existing assembly instance.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `instance_name` | An Existing Assembly Instance | Specifies the assembly instance to modify. |
| `new_instance_name` | A New Assembly Instance | Specifies a new name for the assembly instance. |
| `comments` | String | Specifies comments for the object being created or modified. |
| `location` | Location | Specifies the new location of the assembly instance. |
| `orientation` | Orientation | Specifies the new orientation of the assembly instance using three rotation angles. |
| `relative_to` | An Existing Model, Part Or Marker | Specifies the coordinate system that location and orientation are with respect to. |
