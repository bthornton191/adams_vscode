# interface wizard page modify

Allows you to modify the properties of an existing Adams View wizard page.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `wizard_page_name` | An Existing Wizard Page | Specifies the name of the wizard page to modify. |
| `new_wizard_page_name` | A New Wizard Page | Specifies a new name for the wizard page. |
| `title` | String | Specifies the title displayed on the wizard page. |
| `is_accessible` | Boolean | Specifies whether the page is accessible to the user. |
| `next_commands` | String | Specifies Adams View commands to execute when the Next button is clicked on this page. |
| `back_commands` | String | Specifies Adams View commands to execute when the Back button is clicked on this page. |
| `enable_next` | Boolean | Specifies whether the Next button is enabled on this page. |
| `enable_back` | Boolean | Specifies whether the Back button is enabled on this page. |
| `enabled` | Boolean | Specifies whether the wizard page is enabled. |
| `documentation_text` | String | Specifies the documentation text displayed on the wizard page. |
