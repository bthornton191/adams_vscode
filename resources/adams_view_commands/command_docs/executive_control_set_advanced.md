# executive_control set advanced

Allows you to set the value of an advanced executive control parameter for a model.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `model_name` | An Existing Model | Specifies the model for which the advanced executive control parameter is set. |
| `setting` | String | Specifies the name of the advanced executive control parameter to set. |
| `value` | String | Specifies the value or values to assign to the setting. |
