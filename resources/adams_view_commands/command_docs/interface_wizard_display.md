# interface wizard display

Allows you to display an Adams View wizard dialog.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `wizard_name` | An Existing Wizard | Specifies the name of the wizard to display. |
| `wizard_page_name` | An Existing Wizard Page | Specifies the name of the initial page to display when the wizard is opened. |
| `parameters` | String | Specifies parameter values to pass to the wizard when it is displayed. |
