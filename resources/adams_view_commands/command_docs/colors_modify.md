# colors modify

Allows you to modify an existing custom color definition.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `color_name` | An Existing Color | Specifies the name of the existing color to modify. |
| `new_color_name` | A New Color | Specifies a new name for the color. |
| `red_component` | Real | Specifies the red component of the color, from 0.0 (none) to 1.0 (full). |
| `green_component` | Real | Specifies the green component of the color, from 0.0 (none) to 1.0 (full). |
| `blue_component` | Real | Specifies the blue component of the color, from 0.0 (none) to 1.0 (full). |
| `gradient` | Boolean | Specifies whether the color definition includes a gradient. |
