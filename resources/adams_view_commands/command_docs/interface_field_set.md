# interface field set

Allows you to set the execution parameters and the editing the contents of a field. A field is a space on a dialog box where a data elements can be entered and displayed.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `field _name` | An Existing GI_field | Specifies the name of an existing field |
| `strings` | String | Specifies the strings to be added to the field |
| `database_field` | String | Specifies the database_filed to be added |
| `action` | Field_action | Specifies whether the new addition will replace or add to the existing contents |
| `execute` | Boolean | Sets the execution of the command to ‘yes’ or ‘no’ |
