# assembly attributes

Allows you to set the display attributes of an assembly instance.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `instance_name` | An Existing Assembly Instance | Specifies the assembly instance whose display attributes are to be modified. |
| `scale_of_icons` | Real | Specifies a scale factor applied to the icon size. |
| `size_of_icons` | Length | Specifies the absolute icon size. |
| `visibility` | On/Off | Specifies whether the assembly instance is visible. |
| `name_visibility` | On/Off | Specifies whether the name label of the assembly instance is visible. |
| `transparency` | Integer | Specifies the transparency level of the assembly instance, from 0 (opaque) to 100 (fully transparent). |
| `lod` | Real | Specifies the level-of-detail threshold controlling when simplified graphics are used. |
| `color` | An Existing Color | Specifies the display color of the assembly instance. |
| `entity_scope` | String | Specifies the scope of entities affected by the attribute changes. |
| `active` | On/Off | Specifies whether the assembly instance is active in the simulation. |
| `dependents_active` | On/Off | Specifies whether dependent entities of the assembly instance are active. |
