# xy_plots template delete hlabel

Allows you to delete the horizontal axis label and removes it from the XY plot. Once deleted, this label cannot be retrieved and must be recreated. Note that if there are multiple horizontal axes, the label is deleted from the primary horizontal axis.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `plot_name` | Existing Plot | Specifies a string that identifies an existing plot. |
