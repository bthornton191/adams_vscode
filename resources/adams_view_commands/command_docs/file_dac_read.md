# file dac read

Allows you to import DAC files.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `file_name` | String | Specifies the name of the file that is to be read, written, or executed. |
| `dac_object_name` | Dac_file_object_name | Specifies the name of the dac_file object that will contain the data from the file. Default is the file name. |
