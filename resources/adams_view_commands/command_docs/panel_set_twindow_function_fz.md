# panel set twindow_function fz

Configures the panel test window function to return the Z-axis force between two markers, analogous to the Adams FZ run-time function.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `i_marker_name` | Object | Name of the I (action) marker. |
| `j_marker_name` | Object | Name of the J (reaction) marker. |
| `r_marker_name` | Object | Name of the reference marker for expressing the force. |
