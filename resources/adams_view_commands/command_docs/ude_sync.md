# ude sync

Copies the parameter settings from one UDE instance to another so, when done, the two instances are identical.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `instance_name` | An Existing UDE Instance | Specifies the name of an existing UDE instance whose parameter settings will get modified. |
| `from` | An Existing UDE Instance | Specifies the name of an existing UDE instance whose parameter settings will be applied to the UDE instance specified in instance_name. |
