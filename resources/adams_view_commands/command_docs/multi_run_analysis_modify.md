# multi_run_analysis modify

Modifies an existing multi-run analysis definition.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `multi_run_analysis_name` | Object | Name of the multi-run analysis to modify. |
| `new_multi_run_analysis_name` | String | New name for the multi-run analysis. |
| `comments` | String | Updated descriptive comments for the analysis. |
