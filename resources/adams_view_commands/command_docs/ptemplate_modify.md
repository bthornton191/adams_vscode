# ptemplate modify

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `ptemplate_name` | An Existing Ptemplate | Modifies a string of characters that identifies an existing plot |
| `type / size` | Hardcopy_paper | Hardcopy_paper |
| `orientation` | Hardcopy_orient | Hardcopy_orient |
| `height` | Real | Modifies the height of the info window |
| `width` | Real | Modifies the width of the info window |
