# panel set twindow_function _cosine_fourier_series

Configures the panel test window function to use a cosine Fourier series expression.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `x` | Real | The independent variable expression. |
| `shift` | Real | Time shift (offset) in the function argument. |
| `angular_shift` | Real | Angular shift in the function argument (in radians). |
| `frequency` | Real | Fundamental frequency of the series. |
| `coefficients` | Array | Array of Fourier cosine coefficients. |
| `angular_coefficients` | Array | Array of Fourier cosine coefficients specified in angular frequency. |
