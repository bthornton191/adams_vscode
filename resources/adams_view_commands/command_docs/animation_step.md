# animation step

Allows you to step the animation either forward or backwards as needed.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `direction` | Animation_direction | Specifies whether you want the animation to step in the forward or backward direction |
| `animation_name` | An Existing Animation | Specifies an existing animation name that is to be run |
| `page_name` | An Existing Page | Specifies the page in the Adams Postprocessor that is to be run |
