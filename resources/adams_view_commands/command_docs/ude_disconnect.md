# ude disconnect

Disconnects the output parameters of one User Defined Element (UDE) instance to the input parameters of another.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `instance_name` | An Existing UDE Instance | Specifies the name of an existing UDE whose output parameters will be disconnected from the input parameters of the UDE specified for from_instance_name. |
| `from_instance_name` | An Existing UDE Instance | Specifies the name of an existing UDE whose input parameters will be disconnected from the output parameters of the UDE specified for instance_name. |
