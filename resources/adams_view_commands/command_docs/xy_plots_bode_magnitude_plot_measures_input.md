# xy_plots bode magnitude_plot measures_input

Allows you to define a Bode plot (phase, magnitude, or both, depending on the command) using predefined or user-defined measures of model input and output to define the system.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `plot_name` | Existing Plot | Specifies the name of the Bode plot to be updated |
| `page_name` | Existing Page | Specifies the name of the Bode page that the plot is on. |
| `first_measure` | Existing Measure | Specifies the predefined or user-defined measure of the system input. |
| `second_measure` | Existing Measure | Specifies the predefined or user-defined measure of the system output. |
| `run_name` | Existing Analysis | Specifies the analysis name for the measure results. |
