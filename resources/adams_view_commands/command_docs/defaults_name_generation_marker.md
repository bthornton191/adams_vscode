# defaults name_generation marker

Allows you to set the name-generation prefix for markers.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `marker_prefix` | String | Specifies the text string to be used as a prefix when Adams View generates names for a marker. |
| `floating_marker_prefix` | String | Specifies the text string to be used as a prefix when Adams View generates names for a floating_marker. |
