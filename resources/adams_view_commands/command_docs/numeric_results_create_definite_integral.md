# numeric_results create definite_integral

Allows you to calculate the definite integral of an existing result set component.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `result_set_component_names` | An Existing Component | Identifies two result-set components to perform the operation on. |
| `new_result_set_component_name` | A New Component | Allows you to identify where the new data components are to be stored. |
| `units` | Units_Type_With_Calc | Allows you to specify the type of units to be used for the new result set component. |
