# list_info constraint

Lists information about the current state of the constraint in the system.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `constraint_name` | Existing constraint | Specify an existing constraint name. |
| `joint_name` | Existing joint | Specify an existing joint name. |
| `gear_name` | Existing gear | Specify an existing gear name. |
| `coupler_name` | Existing coupler | Specify an existing coupler name. |
| `jprim_name` | Existing jprim | Specify an existing jprim name. |
| `point_curve_name` | Existing point curve | Specify an existing point curve name. |
| `curve_curve_name` | Existing curve-curve | Specify an existing curve-curve name. |
| `motion_name` | Existing motion | Specify an existing motion name. |
| `user_constraint_name` | Existing user constraint | Specify an existing user constraint name. |
| `brief` | On/Off | Specify whether you want the constraint information in brief or not. |
| `write_to_terminal` | On/Off | Specify whether you want the constraint information to be displayed on the terminal. |
| `file_name` | String | Specify filename in which you want to save the constraint information. |
