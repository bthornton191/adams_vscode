# simulation script copy

Allows you to copy an existing simulation script.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `sim_script_name` | An Existing Sim_script | Specifies the name of the original simulation script which is to be renamed |
| `new_sim_script_name` | A New Sim_script | Specifies the New script name for the copy of the simulation script |
