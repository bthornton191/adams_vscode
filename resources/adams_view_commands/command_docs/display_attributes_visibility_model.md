# display_attributes visibility model

Allows you to set the visibility of an existing model in Adams View.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `model_name` | An Existing Model | Specifies the model to be modified. You use this parameter to identify the existing model to be affected with this command. |
| `visibility` | On_Off_With_Toggle | Specifies the visibility of graphic entities. |
| `name_visibility` | On_Off_With_Toggle | The NAME_VISIBILITY parameter provides control over the visibility of the view name displayed at the top center position of a given view. |
