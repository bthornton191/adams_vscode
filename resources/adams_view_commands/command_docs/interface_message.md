# interface message

Allows the setting the mode of display in plot panel in the plot window

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `graphic_window_level` | Msg_gui_level | Specifies the message type displayed for graphic window event |
| `command_window_level` | Msg_gui_level | Specifies the message type displayed for command window event |
| `start_up_level` | Msg_gui_level | Specifies the message type displayed for start up |
| `threshold_severity` | Severity | Specifies the threshold limit above which a message is displayed |
| `message` | String | Specifies the message dtring to be displayed |
| `enabled` | Boolean | Enables the message setting |
