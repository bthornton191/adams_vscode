# measure create query

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `measure_name` | New name for the measure | Specifies name for the new measure to be created |
| `gui_driver_name` | Existing graphic user interface | Specifies an existing graphic user interface |
| `delay_time` | Time | Specifies the time delay |
| `initial_condition` | Real | Specifies the initial condition |
| `create_measure_display` | Yes/No | Specifies yes if you want to display the strip chart |
| `legend` | String | Specifies the text that will appear at the top of the measure dialog. |
