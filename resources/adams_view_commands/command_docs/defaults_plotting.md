# defaults plotting

The defaults plotting command is used to set the default display properties for XY plots including curve appearance, axis configuration, and grid settings.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `spec_line_color` | An Existing Color | Specifies the default color for specification lines on plots. |
| `spec_line_thickness` | Real | Specifies the default thickness for specification lines. |
| `spec_line_line_type` | String | Specifies the default line type for specification lines. |
| `curve_auto_thickness` | On/Off | Specifies whether curve thickness is assigned automatically. |
| `curve_thickness` | Real | Specifies the default curve thickness. |
| `curve_auto_color` | On/Off | Specifies whether curve color is assigned automatically. |
| `curve_color` | An Existing Color | Specifies the default curve color. |
| `curve_auto_line_type` | On/Off | Specifies whether curve line type is assigned automatically. |
| `curve_line_type` | String | Specifies the default curve line type. |
| `curve_auto_symbol` | On/Off | Specifies whether symbols on curves are assigned automatically. |
| `curve_symbol` | String | Specifies the default symbol type for curves. |
| `curve_frozen` | On/Off | Specifies whether curves are frozen (not updated when data changes) by default. |
| `curve_increment_symbol` | On/Off | Specifies whether symbols are incremented automatically along the curve. |
| `haxis_type` | String | Specifies the default horizontal axis type. |
| `haxis_autoscale` | On/Off | Specifies whether horizontal axis autoscaling is enabled by default. |
| `haxis_alt_autoscale` | On/Off | Specifies whether alternate autoscaling is enabled for the horizontal axis. |
| `haxis_default_limits` | On/Off | Specifies whether default limits are used for the horizontal axis. |
| `haxis_limits` | Real | Specifies the default horizontal axis limits. |
| `haxis_label_offset` | Real | Specifies the offset of the horizontal axis label. |
| `haxis_autodivs` | On/Off | Specifies whether the number of horizontal axis divisions is assigned automatically. |
| `haxis_divs` | Integer | Specifies the default number of horizontal axis divisions. |
| `haxis_increment` | Real | Specifies the default horizontal axis tick mark increment. |
| `haxis_minor_divs` | Integer | Specifies the default number of minor divisions on the horizontal axis. |
| `haxis_placement` | String | Specifies the default placement of the horizontal axis. |
| `vaxis_type` | String | Specifies the default vertical axis type. |
| `vaxis_autoscale` | On/Off | Specifies whether vertical axis autoscaling is enabled by default. |
| `vaxis_alt_autoscale` | On/Off | Specifies whether alternate autoscaling is enabled for the vertical axis. |
| `vaxis_default_limits` | On/Off | Specifies whether default limits are used for the vertical axis. |
| `vaxis_limits` | Real | Specifies the default vertical axis limits. |
| `vaxis_label_offset` | Real | Specifies the offset of the vertical axis label. |
| `vaxis_autodivs` | On/Off | Specifies whether the number of vertical axis divisions is assigned automatically. |
| `vaxis_divs` | Integer | Specifies the default number of vertical axis divisions. |
| `vaxis_increment` | Real | Specifies the default vertical axis tick mark increment. |
| `vaxis_minor_divs` | Integer | Specifies the default number of minor divisions on the vertical axis. |
| `vaxis_placement` | String | Specifies the default placement of the vertical axis. |
| `axis_numbers_font_size` | Real | Specifies the default font size for axis numbers. |
| `axis_label_font_size` | Real | Specifies the default font size for axis labels. |
| `grid_color` | An Existing Color | Specifies the default grid line color. |
| `grid_line_type` | String | Specifies the default grid line type. |
| `grid_thickness` | Real | Specifies the default grid line thickness. |
| `secondary_grid_color` | An Existing Color | Specifies the default secondary grid line color. |
| `secondary_grid_line_type` | String | Specifies the default secondary grid line type. |
| `secondary_grid_thickness` | Real | Specifies the default secondary grid line thickness. |
| `auto_graph_area` | On/Off | Specifies whether the plot graph area is automatically sized. |
| `dependent_axis_type` | String | Specifies the default dependent axis type. |
