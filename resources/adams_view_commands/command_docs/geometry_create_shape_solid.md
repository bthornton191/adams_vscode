# geometry create shape solid

Allows you to create a solid geometry object from one or more solid shell geometries.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `solid_name` | A New Solid | Specifies the name of the new solid. |
| `comments` | String | Specifies comments for the object being created or modified. |
| `shells` | An Existing Solid Shell | Specifies one or more solid shell objects that define the boundary of the solid. |
| `reference_marker` | An Existing Marker | Specifies the marker that defines the coordinate system for the solid geometry. |
