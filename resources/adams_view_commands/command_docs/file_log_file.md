# file log_file

The log_file is an ASCII file Adams View creates, that becomes a record of Adams View session.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `file_name` | String | Specifies the name of the file that is to be read, written, or executed. |
| `write_switch` | On_Off | This parameter controls whether or not Adams View generates a log file. |
| `messages_include` | On_Off | Allows you to specify the inclusion or exclusion of diagnostic messages, produced by Adams View, into the log file |
| `commands_include` | On_Off | COMMANDS_INCLUDE is a parameter to the "file log_file" command |
| `expand_commands` | On_Off | EXPAND_COMMANDS is a parameter on the "file log_file" command |
