# display_attributes color part

Allows you to control the color of graphic entities associated with a part.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `model_name` | An Existing Part | Specifies the part to be modified. You use this parameter to identify the existing part to be affected with this command. |
| `equation_name` | An Existing Equ | Specifies the equation to be modified. You use this parameter to identify the existing equation to be affected with this command. |
| `color` | An Existing Color | Specifies the color the modeling entity should be drawn in. |
| `entity_scope` | Color_Scope | The ENTITY_SCOPE parameter is used to control how a color modification is to affect a particular graphic entity. |
