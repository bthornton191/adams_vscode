# display_attributes visibility marker

Allows you to set the visibility of an existing marker in Adams View.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `marker_name` | An Existing Marker | Specifies the marker to be modified. You use this parameter to identify the existing marker to be affected with this command. |
| `visibility` | On_off_with_toggle | Specifies the visibility of graphic entities. |
| `name_visibility` | On_off_with_toggle | The NAME_VISIBILITY parameter provides control over the visibility of the view name displayed at the top center position of a given view. |
