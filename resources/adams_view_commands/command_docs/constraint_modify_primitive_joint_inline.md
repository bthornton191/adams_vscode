# constraint modify primitive_joint inline

Allows the modification of an inline joint primitive.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `jprim_name` | An Existing Primitive Joint | Specifies the name of an existing primitive joint. |
| `new_jprim_name` | A New Primitive Joint | Specifies a new name for the primitive joint. |
| `adams_id` | Integer | Specifies an integer used to identify this element in the Adams data file. |
| `comments` | String | Specifies comments for the object being created or modified. |
| `i_marker_name` | An Existing Marker | Specifies a marker on the first of two parts connected by this joint. |
| `j_marker_name` | An Existing Marker | Specifies a marker on the second of two parts connected by this joint. |
