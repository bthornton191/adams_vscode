# file binary write

Allows you to writes an Adams View binary database file.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `file_name` | String | Specifies the name of the file that is to be read, written, or executed. |
| `demo_version` | Real | Specifies that the binary file to be written should be for use with the Mechanical Dynamics, Inc. "Adams Demonstration System". |
| `alert_if_exists` | Boolean | Specifies whether or not to issue an alert before writing the file, if a file with the same name already exists |
