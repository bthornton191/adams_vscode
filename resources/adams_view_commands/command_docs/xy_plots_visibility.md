# xy_plots visibility

Controls the visibility of decorative elements on an XY plot.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `plot_name` | Object | Name of the XY plot to modify. |
| `grid_lines` | Boolean | Whether to display grid lines. |
| `tic_marks` | Boolean | Whether to display tick marks on axes. |
| `legend` | Boolean | Whether to display the curve legend. |
| `axis_visibility` | Boolean | Whether to display the plot axes. |
| `axis_labels` | Boolean | Whether to display axis labels. |
| `symbol` | String | Symbol style for data points. |
