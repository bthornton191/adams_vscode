# data_element copy

Allows you to create a replica DATA_ELEMENT within the same model.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `data_element_name` | An Existing Var | Specifies an existing data_element. |
| `new_data_element_name` | A New Var | Specifies the name of the new data_element. You may use this name later to refer to this data_element. |
