# output_control copy request

Allows you to create a replica request. This replica request will be an exact copy of the original with the exception of the name.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `request_name` | String | Specifies an existing request. You may identify an mrequest by typing its name. |
| `new_request_name` | String | Specifies the name of the new request. You may use this name later to refer to this mrequest. Adams View will not allow you to have two mrequests with the same name, so you must provide a unique name. |
