# defaults page_attributes

The defaults page_attributes command is used to set default header and footer content and layout for Adams View pages.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `header_left_text` | String | Specifies the text displayed in the left header area. |
| `header_left_font_size` | Real | Specifies the font size for the left header text. |
| `header_left_color` | An Existing Color | Specifies the color for the left header text. |
| `header_left_image_file` | String | Specifies the image file displayed in the left header area. |
| `header_left_image_height` | Real | Specifies the height of the image in the left header area. |
| `header_center_text` | String | Specifies the text displayed in the center header area. |
| `header_center_color` | An Existing Color | Specifies the color for the center header text. |
| `header_center_font_size` | Real | Specifies the font size for the center header text. |
| `header_center_image_file` | String | Specifies the image file displayed in the center header area. |
| `header_center_image_height` | Real | Specifies the height of the image in the center header area. |
| `header_right_text` | String | Specifies the text displayed in the right header area. |
| `header_right_font_size` | Real | Specifies the font size for the right header text. |
| `header_right_color` | An Existing Color | Specifies the color for the right header text. |
| `header_right_image_file` | String | Specifies the image file displayed in the right header area. |
| `header_right_image_height` | Real | Specifies the height of the image in the right header area. |
| `footer_left_text` | String | Specifies the text displayed in the left footer area. |
| `footer_left_font_size` | Real | Specifies the font size for the left footer text. |
| `footer_left_color` | An Existing Color | Specifies the color for the left footer text. |
| `footer_left_image_file` | String | Specifies the image file displayed in the left footer area. |
| `footer_left_image_height` | Real | Specifies the height of the image in the left footer area. |
| `footer_center_text` | String | Specifies the text displayed in the center footer area. |
| `footer_center_color` | An Existing Color | Specifies the color for the center footer text. |
| `footer_center_font_size` | Real | Specifies the font size for the center footer text. |
| `footer_center_image_file` | String | Specifies the image file displayed in the center footer area. |
| `footer_center_image_height` | Real | Specifies the height of the image in the center footer area. |
| `footer_right_text` | String | Specifies the text displayed in the right footer area. |
| `footer_right_font_size` | Real | Specifies the font size for the right footer text. |
| `footer_right_color` | An Existing Color | Specifies the color for the right footer text. |
| `footer_right_image_file` | String | Specifies the image file displayed in the right footer area. |
| `footer_right_image_height` | Real | Specifies the height of the image in the right footer area. |
| `page_template_file` | String | Specifies the name of a file containing a page template to apply. |
| `layout` | String | Specifies the default page layout mode. |
| `color` | An Existing Color | Specifies the default background color for the page. |
