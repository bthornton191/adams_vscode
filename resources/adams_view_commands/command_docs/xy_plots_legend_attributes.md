# xy_plots legend attributes

Allows you to modify the attributes of a legend, including its border color, line style, and thickness.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `legend_name` | An Existing Legend | Specifies name of the legend to modify. |
| `colo` | An Existing Color | Specifies the COLOR of a graphic object |
| `line_type` | Line_style | This parameter allows the selection of the line type for a curve. |
| `thickness` | Real | This parameter allows the specification of the thickness of the curve. |
