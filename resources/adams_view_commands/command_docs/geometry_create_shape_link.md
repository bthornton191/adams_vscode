# geometry create shape link

Allows you to create a link geometry object between two markers.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `link_name` | A New Link | Specifies the name of the new link. |
| `comments` | String | Specifies comments for the object being created or modified. |
| `i_marker` | An Existing Marker | Specifies the marker at the first end (I end) of the link. |
| `j_marker` | An Existing Marker | Specifies the marker at the second end (J end) of the link. |
| `width` | Length | Specifies the width of the link cross-section. |
| `depth` | Length | Specifies the depth (thickness) of the link cross-section. |
