# defaults graphics

The defaults graphics command is used to set the default rendering and display settings for the Adams View graphics window.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `light_intensity` | Real | Specifies the default intensity of the light source used for rendering. |
| `ambient_light` | Real | Specifies the default ambient light level applied to the scene. |
| `camera_angle` | Angle | Specifies the default camera field-of-view angle. |
| `pick_tolerance` | Real | Specifies the pixel tolerance used when picking objects in the graphics window. |
| `caching` | On/Off | Specifies whether graphics caching is enabled. |
| `memory_caching` | On/Off | Specifies whether display list memory caching is used. |
| `cache_data_compression` | On/Off | Specifies whether data compression is applied to cached graphics. |
| `lighting` | On/Off | Specifies whether lighting calculations are applied during rendering. |
| `reflections` | On/Off | Specifies whether reflections are rendered. |
| `specular` | On/Off | Specifies whether specular highlights are rendered. |
| `shininess` | Real | Specifies the shininess (specular exponent) used for surface highlight rendering. |
| `transparent_quality` | String | Specifies the quality level for rendering transparent objects. |
