# xy_plots bode phase_plot result_set_input

Creates a Bode phase plot from two result set inputs.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `plot_name` | String | Name for the Bode phase plot. |
| `page_name` | String | Name of the page on which to place the plot. |
| `first_resset` | Object | Name of the first result set (numerator transfer function). |
| `second_resset` | Object | Name of the second result set (denominator transfer function). |
