# interface record

Allows you to start or stop recording Adams View commands to a file.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `record` | On/Off | Specifies whether recording is started or stopped. |
| `comments` | String | Specifies comments to include at the beginning of the recorded command file. |
| `file_name` | String | Specifies the name of the file to which the commands are recorded. |
