# panel set twindow_function dz

Configures the panel test window function to return the Z-axis displacement between two markers, analogous to the Adams DZ run-time function.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `i_marker_name` | Object | Name of the I (from) marker. |
| `j_marker_name` | Object | Name of the J (to) marker. |
| `r_marker_name` | Object | Name of the reference marker for expressing the result. |
