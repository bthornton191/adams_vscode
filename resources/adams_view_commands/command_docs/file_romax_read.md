# file romax read

Allows you to import a Romax gearbox model file into Adams View.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `file_name` | String | Specifies the name of the Romax file to read. |
| `model_name` | An Existing Model | Specifies the model into which the Romax data is imported. |
| `location` | Location | Specifies the location at which to place the imported Romax model. |
| `orientation` | Orientation | Specifies the orientation of the imported Romax model using three rotation angles. |
| `relative_to` | An Existing Model, Part Or Marker | Specifies the coordinate system that the location and orientation are with respect to. |
