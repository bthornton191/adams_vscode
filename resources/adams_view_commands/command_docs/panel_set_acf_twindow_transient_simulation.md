# panel set acf_twindow transient_simulation

Configures transient simulation parameters for an ACF test window panel.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `number_of_steps` | Integer | Number of output steps for the simulation. |
| `step_size` | Real | Size of each time step. |
| `end_time` | Real | End time for the simulation. |
| `duration` | Real | Total duration of the simulation. |
