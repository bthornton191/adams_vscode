# file adams_data_set read

Allows you to read an Adams data set file (.adm) into Adams View.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `file_name` | String | Specifies the name of the Adams data set file to read. |
| `model_name` | A New Model | Specifies the name to assign to the model read from the file. |
| `length` | Linear Units | Specifies the length units in the file if different from the current default. |
| `force` | Force Units | Specifies the force units in the file if different from the current default. |
| `mass` | Mass Units | Specifies the mass units in the file if different from the current default. |
| `time` | Time Units | Specifies the time units in the file if different from the current default. |
