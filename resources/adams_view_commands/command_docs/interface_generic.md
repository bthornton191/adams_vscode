# interface generic

Allows you to send generic interface commands with arbitrary integer, real, and string arguments.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `ints` | Integer | Specifies integer arguments passed to the generic interface command. |
| `options` | String | Specifies option flags for the generic interface command. |
| `reals` | Real | Specifies real-valued arguments passed to the generic interface command. |
| `strings` | String | Specifies string arguments passed to the generic interface command. |
| `keys` | String | Specifies key arguments passed to the generic interface command. |
