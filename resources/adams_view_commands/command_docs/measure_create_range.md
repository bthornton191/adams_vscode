# measure create range

Allows you to use range measures to obtain statistical feedback about any existing measure. Ranges dynamically calculate the maximum, minimum, average, or variation characteristics of any measure.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `measure_name` | New Name For The Range Measure | Specifies the name of the new measure to be created. |
| `type` | Minimum, Maximum, Average, Variation, Same_as | Specifies type of the range measure. |
| `of_measure_name` | Existing Measure | Specifies existing, predefined measure to analyze. |
| `create_measure_display` | Yes/no | Type Yes if you want to display the strip chart of the measure. |
| `legend` | String | Specifies comments for the range measure. |
