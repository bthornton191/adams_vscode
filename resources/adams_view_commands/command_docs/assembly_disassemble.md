# assembly disassemble

Allows you to disassemble an assembly instance, dissolving it into its constituent objects.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `instance_name` | An Existing Assembly Instance | Specifies the assembly instance to disassemble. |
| `top_level_only` | Boolean | Specifies whether to disassemble only the top level of the assembly without recursively disassembling nested sub-assemblies. |
