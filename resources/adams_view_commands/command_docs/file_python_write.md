# file python write

Allows you to write an Adams View model to a Python script file.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `entity_name` | An Existing Model | Specifies the model or entity to export as a Python script. |
| `file_name` | String | Specifies the name of the Python file to write. |
| `use_parasolid` | String | Specifies whether to write geometry using the Parasolid format or the Adams View command language format. |
