# output_control copy mrequest

Allows you to create a replica mrequest for a model. This mrequest is will be an exact copy of the original except the name. You may modify the replica without any harm done to the original.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `mrequest_name` | String | Specifies an existing mrequest. You may identify an mrequest by typing its name. |
| `new_mrequest_name` | String | Specifies the name of the new mrequest. You may use this name later to refer to this mrequest. Adams View will not allow you to have two mrequests with the same name, so you must provide a unique name. |
