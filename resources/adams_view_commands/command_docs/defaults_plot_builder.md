# defaults plot_builder

The defaults plot_builder command is used to set default options for the Adams View plot builder.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `auto_axis` | On/Off | Specifies whether the plot builder automatically assigns axes when adding curves. |
| `ude_measures` | On/Off | Specifies whether user-defined element measures are included in the plot builder's measure list. |
