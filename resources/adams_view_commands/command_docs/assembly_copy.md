# assembly copy

Allows you to copy an assembly definition and/or instance.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `definition_name` | An Existing Assembly Definition | Specifies the assembly definition to copy. |
| `new_definition_name` | A New Assembly Definition | Specifies the name for the new assembly definition. |
| `instance_name` | An Existing Assembly Instance | Specifies the assembly instance to copy. |
| `new_instance_name` | A New Assembly Instance | Specifies the name for the new assembly instance. |
