# panel set twindow_function tx

Configures the panel test window function to return the X-axis displacement (translational) between two markers, analogous to the Adams TX run-time function.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `i_marker_name` | Object | Name of the I (from) marker. |
| `j_marker_name` | Object | Name of the J (to) marker. |
| `r_marker_name` | Object | Name of the reference marker for expressing the result. |
