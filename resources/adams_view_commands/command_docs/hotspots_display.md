# hotspots display

Allows you to display hotspots. Hot spots are locations of high stress or strain on a flexible body or rigid stress object. You can easily locate and view hot-spot information during animation displays in the Adams PostProcessor. When the Adams Durability plugin is loaded, a Hot Spots tab is available on the Adams PostProcessor dashboard for Animation displays. This tab allows you to define the hot spots and control their display.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `animation_name` | Existing Animation | Specifies the animation on which the hotspot should be displayed. |
| `visibility` | On/off | Sets the visibility of the hotspot to ON or OFF. |
