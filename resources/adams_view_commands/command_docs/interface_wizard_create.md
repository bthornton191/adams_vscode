# interface wizard create

Allows you to create an Adams View wizard dialog.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `wizard_name` | A New Wizard | Specifies the name of the new wizard. |
| `enabled` | Boolean | Specifies whether the wizard is enabled when first displayed. |
| `help_text` | String | Specifies the help text associated with the wizard. |
| `documentation_text` | String | Specifies the documentation text displayed in the wizard. |
| `units` | String | Specifies the units system used in the wizard fields. |
| `horiz_resizing` | String | Specifies the horizontal resizing behavior of the wizard. |
| `vert_resizing` | String | Specifies the vertical resizing behavior of the wizard. |
| `location` | Location | Specifies the screen location at which the wizard is displayed. |
| `height` | Real | Specifies the height of the wizard window. |
| `width` | Real | Specifies the width of the wizard window. |
| `title` | String | Specifies the title bar text for the wizard window. |
| `page_count` | Integer | Specifies the number of pages in the wizard. |
| `icon_label` | String | Specifies the icon label displayed in the wizard header. |
| `start_commands` | String | Specifies Adams View commands to execute when the wizard is started. |
| `finish_commands` | String | Specifies Adams View commands to execute when the wizard is finished. |
| `execution_commands` | String | Specifies Adams View commands to execute when the wizard is executed. |
| `enable_finish` | Boolean | Specifies whether the Finish button is enabled. |
| `decorate` | Boolean | Specifies whether standard window decorations are applied to the wizard. |
| `resizable` | Boolean | Specifies whether the wizard window is resizable. |
| `width_minimum` | Real | Specifies the minimum width of the wizard window. |
| `width_maximum` | Real | Specifies the maximum width of the wizard window. |
| `height_minimum` | Real | Specifies the minimum height of the wizard window. |
| `height_maximum` | Real | Specifies the maximum height of the wizard window. |
| `grab_all_input` | Boolean | Specifies whether the wizard captures all mouse and keyboard input while displayed. |
| `iconifiable` | Boolean | Specifies whether the wizard window can be iconified (minimized). |
