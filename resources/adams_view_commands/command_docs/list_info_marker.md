# list_info marker

Lists information about the current state of the markers in the system.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `marker_name` | Existing marker | Specify the marker name you want information to be displayed about. |
| `brief` | On/Off | Specify whether you want the marker information in brief or not. If brief is turned on, only location and orientation are displayed without dependants information. |
| `write_to_terminal` | On/Off | Specify whether you want the marker information to be displayed on the terminal. |
| `file_name` | String | Specify filename in which you want to save the marker information. |
