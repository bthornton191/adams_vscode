# simulation configuration save

Allows you to store the current Adams model and simulation conditions, or only the current simulation states. This allows the user to return to the model or states at a later time, using the RELOAD command.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `model_name` | An Existing Model | Specifies an existing model. |
| `configuration_label` | String | Enter a string for the configuration label |
| `analysis_included` | Boolean | Takes the Boolean values of YES or NO depending on whether the analysis has to be included with the configuration or not. |
