# contour_plot legend delete

Allows you to delete a legend in the animation dashboard for a contoured plot of deformation, stress/strain or modal forces while in Adams Flex, Adams Vibration or Adams Durability.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `legend_name` | An Existing Contour_plot Legend | Specifies the name of an existing contour_plot legend |
