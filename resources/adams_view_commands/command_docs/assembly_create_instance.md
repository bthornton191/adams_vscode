# assembly create instance

Allows you to create an instance of an existing assembly definition.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `instance_name` | A New Assembly Instance | Specifies the name of the new assembly instance. |
| `definition_name` | An Existing Assembly Definition | Specifies the assembly definition to instantiate. |
| `comments` | String | Specifies comments for the object being created or modified. |
| `location` | Location | Specifies the location at which to place the assembly instance. |
| `orientation` | Orientation | Specifies the orientation of the assembly instance using three rotation angles. |
| `relative_to` | An Existing Model, Part Or Marker | Specifies the coordinate system that location and orientation are with respect to. |
