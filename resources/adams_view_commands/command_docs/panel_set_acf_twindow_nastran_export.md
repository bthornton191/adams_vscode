# panel set acf_twindow nastran_export

Configures Nastran export settings for an ACF test window panel.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `type` | String | Type of Nastran export (e.g., `BDF`, `OP2`). |
| `nastran_output_file_prefix` | String | File name prefix for the Nastran output files. |
| `configuration_file_name` | String | Name of the Nastran configuration file to use. |
