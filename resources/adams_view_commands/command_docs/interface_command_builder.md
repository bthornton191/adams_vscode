# interface command_builder

Allows you to open the Adams View Command Builder interface dialog with an optional initial command prefix and field selection.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `command_prefix` | String | Specifies the command string to pre-populate in the Command Builder. |
| `field_name` | String | Specifies the name of the field in the Command Builder to initially focus. |
| `initial_entity` | String | Specifies an initial entity name to insert into the Command Builder. |
