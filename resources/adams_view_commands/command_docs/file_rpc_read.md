# file rpc read

Allows you to read RPC III files into Adams View.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `file_name` | String | Specifies the name of the file that is to be read, written, or executed. |
| `rpc_object_name` | RPC_file_object_name | Specifies the name of the RPC object that is to hold the RPC data. If unspecified, uses the file name. |
