# numeric_results create values

Allows you to create a result set by specifying the values directly.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `new_result_set_component_name` | A New Component | Allows you to identify where the new data components are to be stored. |
| `values` | Real | Specifies the real values to be stored in the result set component.The values can be a list of numbers, or an expression referencing an array of existing values. |
| `units` | Units_Type_With_Calc | Allows you to specify the type of units to be used for the new result set component. |
