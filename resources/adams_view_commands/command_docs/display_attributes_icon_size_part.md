# display_attributes icon_size part

Allows you to specify the size the Adams View part icon will appear in.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `part_name` | An Exisisting Part | Specifies the part to be modified. You use this parameter to identify the existing part to be affected with this command. |
| `scale_of_icons` | Real | Specifies a unit-less scale factor to be applied to the current icon size. |
| `size_of_icons` | Length | Specifies the size, in modeling units, the Adams View icons will appear in. |
