# output_control copy femdata

Allows you to create a replica of an existing femdata object. This replica will be an exact copy of the original, except for the name.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `femdata_name` | String | Specifies an existing femdata. You identify a femdata by typing its name. |
| `new_femdata_name` | String | Specifies the name of the new femdata. You may use this name later to refer to this femdata. |
