# display_attributes visibility force

Allows you to set the visibility of an existing force in Adams View.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `force_name` | An Existing Force | Specifies the force to be modified. You use this parameter to identify the existing force to be affected with this command. |
| `visibility` | On_off_with_toggle | Specifies the visibility of graphic entities. |
| `name_visibility` | On_off_with_toggle | The NAME_VISIBILITY parameter provides control over the visibility of the view name displayed at the top center position of a given view. |
