# defaults results

The defaults results command is used to set the default behavior for handling simulation results data.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `write_through` | On/Off | Specifies whether result data is written to disk immediately as it is generated, rather than buffered in memory. |
| `discard_output` | On/Off | Specifies whether simulation output is discarded after each run instead of being retained. |
