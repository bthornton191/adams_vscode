# simulation script read_acf

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `sim_script_name` | An Existing Sim_script | Enters the name of the script file that you want to associate with the .acf file read in or to modify an existing script name. |
| `file_name` | String | Specifies the Adams Solver(.acf file) that will be run as an Adams Solver command script. |
