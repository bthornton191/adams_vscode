# assembly replace type_of_instance

Allows you to replace all assembly instances of one type within a parent assembly with instances of a different type.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `instance_type` | String | Specifies the assembly definition type of instances to replace. |
| `parent` | An Existing Assembly Instance | Specifies the parent assembly instance in which the replacement is performed. |
| `with_type` | String | Specifies the replacement assembly definition type. |
