# defaults note_attributes

The defaults note_attributes command is used to set the default display attributes for note objects in Adams View.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `note_scale` | Real | Specifies the default scale factor applied to text in notes. |
| `note_border_color` | An Existing Color | Specifies the default color of the note border. |
| `note_border_style` | String | Specifies the default line style of the note border. |
| `note_border_thickness` | Real | Specifies the default thickness of the note border line. |
| `note_border_fill` | Boolean | Specifies whether note backgrounds are filled by default. |
| `note_alignment` | String | Specifies the default text alignment within notes. |
| `note_rotation` | Angle | Specifies the default rotation angle applied to notes. |
