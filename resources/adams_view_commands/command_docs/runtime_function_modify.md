# runtime_function modify

Allows you to modify an existing runtime function.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `runtime_function_name` | An Existing Runtime Function | Specifies the name of the runtime function to modify. |
| `new_runtime_function_name` | A New Runtime Function | Specifies a new name for the runtime function. |
| `text_of_expression` | String | Specifies the Adams function expression that defines the runtime function. |
| `argument_names` | String | Specifies the names of the arguments accepted by the runtime function. |
