# defaults animation_settings

The defaults animation_settings command is used to set the default animation playback settings.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `frame_delay` | Real | Specifies the delay in milliseconds between animation frames during playback. |
| `contour_plots` | On/Off | Specifies whether contour plots are displayed during animation. |
| `fps_display` | On/Off | Specifies whether the frames-per-second counter is displayed during animation playback. |
| `loop` | On/Off | Specifies whether animation playback loops continuously. |
