# numeric_results create sqrt_sum_of_the_squares

Allows you to create a new result set component from the square root of the sum of squares (RSS) of two existing result set components.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `result_set_component_names` | An Existing Result Set Component | Specifies the two result set components whose values are combined using the RSS formula. |
| `new_result_set_component_name` | A New Result Set Component | Specifies the name of the new result set component created from the RSS calculation. |
| `units` | String | Specifies the units for the new result set component. If set to calculate, the units are derived automatically. |
