# simulation multi_run save

Allows you to save the results of the last simulation under a new name so that you can animate or plot the results at a later time.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `analysis_name` | New Analysis Name | Specifies the name of the analysis to be saved |
| `auto_increment` | Yes/No | Select if you want Adams View to automatically increment the run names when you save subsequent simulations. |
| `title` | String | Specifies a string |
