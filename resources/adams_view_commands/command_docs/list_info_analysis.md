# list_info analysis

Specifies information about the analysis within the model such as the result set, graphics file, result file, request file, parent etc.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `analysis_name` | Existing analysis | Specify an existing analysis name. |
| `brief` | On/Off | Specify whether the information needs to be specified in brief or not. If brief is turned off, it will display the details of all the steps in the internal solver. |
| `write_to_terminal` | On/Off | Specify whether writing the information to the terminal is necessary. |
| `file_name` | String | Specify a location and name of file to which the information needs to be written. |
