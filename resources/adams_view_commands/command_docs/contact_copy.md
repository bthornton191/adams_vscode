# contact copy

Allows you to copy an existing contact and creates an identical new contact with a different name, which is to be specified by the user.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `contact_name` | An Existing Contact | Specifies the contact to be deleted. |
| `new_contact_name` | A New Contact | Specifies the new contact which will be a copy of the existing one. |
