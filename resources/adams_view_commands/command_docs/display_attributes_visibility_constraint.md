# display_attributes visibility constraint

Allows you to set the visibility of an existing constraint in Adams View.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `constraint _name` | An Existing Constraint | Specifies the constraint to be modified. You use this parameter toidentify the existing constraint to be affected with this command. |
| `visibility` | On_off_with_toggle | Specifies the visibility of graphic entities. |
| `name_visibility` | On_off_with_toggle | The NAME_VISIBILITY parameter provides control over the visibility of the view name displayed at the top center position of a given view. |
