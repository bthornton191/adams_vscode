# interface database_navigator

Allows the creating a database_navigator.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `top_object` | An Existing Entity | Specifies the top object, all the objects belonGIng to it will be displayed |
| `name_filter` | String | Specifies the name of a object under top object, all the objects belonGIng to it will be displayed |
| `type_filter` | Ent | Specifies the type of entities which will be displayed |
| `display` | On_off | Specifies the default parameter for displaying database navigator |
| `mode` | Mode_options | Specify the mode of displaying the objects beneath the top_object |
