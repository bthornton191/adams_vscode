# list_info variable

Lists information about variables in the current model.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `variable_name` | Existing variable | Specify an existing variable whose information has to be listed. |
| `brief` | On/Off | Specify whether the variable information required is in brief or not. |
| `write_to_terminal` | On/Off | Specify whether the information has to be displayed on the terminal. |
| `file_name` | String | Specify filename in which the information needs to be written. |
