# file vrml write

Allows you to write an Adams View model or animation to a VRML (Virtual Reality Modeling Language) file for 3D visualization.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `analysis_name` | An Existing Analysis | Specifies an existing analysis whose animated configuration is exported. |
| `model_name` | An Existing Model | Specifies the model to export to the VRML file. |
| `file_name` | String | Specifies the name of the VRML file to write. |
| `controls` | String | Specifies options controlling the VRML output format and content. |
