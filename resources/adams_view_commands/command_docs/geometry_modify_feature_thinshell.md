# geometry modify feature thinshell

Allows you to modify an existing thinshell.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `thin_shell_name` | An Existing Thin_shell | Specifies the name of an existing thin shell to be modified. |
| `new_thin_shell_name` | A New Thin_shell | Specifies the new name for the thinshell being modified. |
| `thickness` | Length | Specifies the thickness of the remaining shell after you hollow the object. |
