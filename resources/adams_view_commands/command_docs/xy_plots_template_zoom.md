# xy_plots template zoom

Allows you to set the horizontal and vertical axis limits of a plot by selecting a rectangular region of a plot.

## Parameters

| Parameter | Type | Description |
|---|---|---|
| `plot_name` | Existing Plot | Specifies a string of characters that identifies one or more existing plots. |
| `x_graph_min` | Real | Right coordinate of zoom region. |
| `x_graph_max` | Real | Left coordinate of zoom region. |
| `y_graph_min` | Real | Bottom coordinate of zoom region. |
| `y_graph_max` | Real | Top coordinate of zoom region. |
