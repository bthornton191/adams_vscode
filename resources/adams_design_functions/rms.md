# RMS

Returns the root mean square of the values.

## Format
```
RMS (Values)
```

## Arguments

**Values**
: The array of values for which to compute the RMS function.

## Example

The following example illustrates the use of the RMS function:

### Function
```
RMS({0,1,4,9,16})
```

### Result
```
(SQRT(MEAN({0,1,4,9,16}**2)))
```
