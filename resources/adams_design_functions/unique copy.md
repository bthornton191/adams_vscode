# UNIQUE

Deletes all duplicate elements from an array.

## Format 
```adams_cmd
UNIQUE(Array) 
```
## Arguments

**Array**
: An array

## Example
The following example illustrates the use of the `UNIQUE` function:

### Function
```adams_cmd
var cre var=UNIQUE("marker")
```

### Result
```adams_cmd
.model_1.ground.MAR
```
