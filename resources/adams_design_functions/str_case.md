# STR_CASE

Returns a string from an input string that has been modified according to an integer value.

## Format
```
STR_CASE (String to Change, Case)
```

## Arguments

**String to Change**
: Text string.

**Case**
: Integer value that determines case type as noted in the list below:■1 = Upper case ■2 = Lower case ■3 = Mixed case ■4 = Sentence case

## Example

The following functions return modified strings, as specified:

### Function
```
STR_CASE("this is a TEST!",1)
```

### Result
```
THIS IS A TEST!
```
