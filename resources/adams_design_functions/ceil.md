# CEIL

Returns the smallest integer greater than x.

## Format 
```adams_cmd
CEIL(x) 
```
## Argument 

**x**
: Any valid expression that evaluates to a real number.  

## Example 

The following example illustrates the use of the `CEIL` function:

### Function  
```adams_cmd
CEIL(10.001)  
```

### Result  
```adams_cmd
11  
```
