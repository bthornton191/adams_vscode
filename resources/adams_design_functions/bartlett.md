# BARTLETT

Apply the Bartlett window to the input array and return the new array.

## Format
```adams_cmd
BARTLETT (a)
```
## Arguments

 



**a** 
: An array.  


## Example 

The following is an illustration of the `BARTLETT` function:

 



### Function  
```adams_cmd
bartlett ({1, 2, 3, 4, 2})  
```

### Result  
```adams_cmd
{0, 1, 3, 2, 0}  
```
