# EXCLUDE

Excludes a value from an array.

## Format
```adams_cmd
EXCLUDE(A, X)
```

## Arguments
 
**A**
: An array.

**X**
: A real number.

## Example
Assume that the array in the following function contains the values 1 through 10:
 
### Function
```adams_cmd
EXCLUDE(.MOD1.A,4)
```

### Result
removes 4 from the list
