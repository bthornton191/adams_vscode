# PARZEN

Returns the 1xN array of values after applying the PARZEN window function.

## Format
```
PARZEN (a)
```

## Arguments

**a**
: 1xN array of real numbers.

## Example

The following example illustrates the use of the PARZEN function:

### Function
```
PARZEN ({1,2,3,4,2})
```

### Result
```
{0.333, 1.333, 3.0, 2.667, 0.667}
```
