# FIRST_N

Returns the first N elements of an array.

## Format
```
FIRST_N (A,N)
```

## Arguments

**A**
: An array.

**N**
: Number of elements to return.

## Example

The following examples illustrate the use of the FIRST_N function:

### Function
```
FIRST_N ({ }, 3)
```

### Result
```
{}
```
