# RECTANGULAR_WINDOW

Generate the RECTANGULAR window.

## Format
```
RECTANGULAR_WINDOW (n)
```

## Arguments

**n**
: An integer value.

## Example

The following example is an illustration of the RECTANGULAR_WINDOW function:

### Function
```
RECTANGULAR_WINDOW (5)
```

### Result
```
{1, 1, 1, 1, 1}
```
