# WELCH

Returns a 1xN array of values after applying the WELCH_WINDOW function.

## Format
```
WELCH (a)
```

## Arguments

**a**
: A 1xN array of real numbers.

## Example

The following example illustrates the use of the WELCH function:

### Function
```
WELCH ({1,2,3,4,2})
```

### Result
```
{0.556, 1.778, 3.0, 3.556, 1.111}
```
