# STACK

Returns the concatenation of two matrixes with the same number of columns.

## Format
```
STACK (M1, M2)
```

## Arguments

**M1**
: A matrix of arbitrary shape.

**M2**
: A matrix with the same number of columns as M1.

## Example

The following example illustrates the use of the STACK function:

### Function
```
STACK({[1,2], [3,4]}, {[1,1], [2,2]})
```

### Result
```
{[1,2,1,1], [3,4,2,2]}
```
