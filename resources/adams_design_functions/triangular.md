# TRIANGULAR

Apply the TRIANGULAR window to the input array and return the new array.

## Format
```
TRIANGULAR (a)
```

## Arguments

**a**
: An array.

## Example

The following example is an illustration of the TRIANGULAR function:

### Function
```
triangular ({1, 2, 3, 4, 2})
```

### Result
```
{0.3333 1.3333 3.0000 2.6667 0.6667}
```
