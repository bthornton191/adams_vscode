# STR_TIMESTAMP

Returns the current date and time in the default format.

## Format
```
STR_TIMESTAMP ()
```

## Arguments

**None**
: No argument is needed; this function returns the current date in the following format:%Y/%m/%d,%H:%M:%S

## Example

The following function returns the current date and time (Jan 5, 1998, is the current date):

### Function
```
STR_TIMESTAMP()
```

### Result
```
1998/01/05,02:30:28
```
