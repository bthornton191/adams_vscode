# READ_T_O_CLOSE_FILE

Closes a previously opened TeimOrbit file. Note that only one TO file can be read from at a time, and this command will close the currently open file. Returns 1 on success, and 0 if any errors occur.

## Format
```
READ_T_O_CLOSE_FILE ()
```

## Arguments

None

## Example

### Function
```
READ_T_O_CLOSE_FILE()
```

### Result
```
1
```
