# RECTANGULAR

Returns the 1xN array of values after applying the RECTANGULAR window function.

## Format
```
rectangular (a)
```

## Arguments

**a**
: An array.

## Example

The following example illustrates the use of the RECTANGULAR function:

### Function
```
RECTANGULAR ({1, 2, 3, 4, 2})
```

### Result
```
{1.0, 2.0, 3.0, 4.0, 2.0}
```
