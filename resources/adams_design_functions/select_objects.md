# SELECT_OBJECTS

Returns an array of selected objects, using the Database Navigator to provide you with selections.

## Format
```
SELECT_OBJECTS (Parent, Wildcard, Type)
```

## Arguments

**Parent**
: A database object defining the scope of the search used to find the displayed objects.

**Wildcard**
: A character string defining a name matching pattern filter for the displayed objects.

**Type**
: A character string defining the type of the object to display.
