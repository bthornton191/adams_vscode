# SERIES

Generates an array based on a start value, an increment value, and an array length.

## Format
```
SERIES (Real, Real, Integer)
```

## Arguments

**Real**
: Start value.

**Real**
: Increment value.

**Integer**
: Array length.

## Example

The following example illustrates the use of the SERIES function:

### Function
```
SERIES(1,2,3)
```

### Result
```
1, 3, 5
```
