# NORM2

Returns the square root of the sum of the squares of the elements of a matrix.

## Format
```
NORM2 (M)
```

## Arguments

**M**
: A matrix.

## Example

The following example illustrates the use of the NORM2 function:

### Function
```
NORM2({[1, 2], [3, 4]})
```

### Result
```
5.47722 (SQRT(1**2 + 2**2 + 3**2 + 4**2))
```
