# TILDE

Returns the TILDE function of an array. You can use TILDE only on 3x1 matrixes.

## Format
```
TILDE (A)
```

## Arguments

**A**
: An array.

## Example

The following example illustrates the use of the TILDE function:

### Function
```
TILDE({x, y, z})
```

### Result
```
{[ 0, -z, y], [z, 0, -x], [-y, x, 0]}
```
