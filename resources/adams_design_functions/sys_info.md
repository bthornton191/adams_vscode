# SYS_INFO

Returns a character string containing information that you requested about the system.

## Format
```
SYS_INFO (info_type)
```

## Arguments

**info_type**
: A character string indicating the type of system information you want returned. Below are five values that you can query: GID - Numeric group id GROUPNAME - Group name HOSTNAME - Host name UID - Numeric user id USERNAME - User name REALNAME - The user's name, if it is known.
