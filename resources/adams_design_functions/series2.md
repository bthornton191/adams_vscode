# SERIES2

Calculates an increment based on start and end values and a given number of increments.

## Format
```
SERIES 2 (Real, Real, Integer)
```

## Arguments

**Real**
: Start value.

**Real**
: End value.

**Integer**
: Number of increments.

## Example

The following example illustrates the use of the SERIES2 function:

### Function
```
SERIES2(2, 8, 4)
```

### Result
```
2, 4, 6, 8
```
