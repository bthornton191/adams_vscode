# SORT_INDEX

Returns the indexes of a matrix sorted in the direction you specified.

## Format
```
SORT_INDEX (M, D)
```

## Arguments

**M**
: A matrix.

**D**
: Direction in which the matrix is sorted: ■a = ascending ■d = descending

## Example

The following example illustrates the use of the SORT_INDEX function:

### Function
```
SORT_INDEX({3,5,4,2},"a")
```

### Result
```
{4,1,3,2}
```
