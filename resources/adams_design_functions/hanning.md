# HANNING

Returns a 1xN array of values after applying the HANNING window function.

## Format
```
HANNING (A)
```

## Arguments

**A**
: A 1xN array of real numbers.

## Example

The following example illustrates the use of the HANNING function:

### Function
```
HANNING ({1,2,3,4,2})
```

### Result
```
{0.0, 1.0, 3.0, 2.0, 0.0}
```
