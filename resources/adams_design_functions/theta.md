# THETA

Returns the second angle associated with a Body 313 rotation sequence from one coordinate system object to another. This second rotation is referred to as the theta,  , angle, and is used in association with the psi, , (1st rotation) and phi,  , (3rd rotation) angles.

## Format
```
THETA (Object, Reference Frame)
```

## Arguments

**Object**
: Coordinate system object whose rotation is being measured.

**Reference Frame**
: Coordinate system object with respect to which the rotation is being measured.
