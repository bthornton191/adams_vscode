# WRITE_T_O_SUBBLOCK

Write subblock to currently selected block in the open file. Returns 1 on success and 0 if errors occur.

## Format
```
WRITE_T_O_SUBBLOCK (SubBlock Name)
```

## Arguments

**SubBlock Name**
: Name of subblock to write to file.

## Example

### Function
```
WRITE_T_O_SUBBLOCK ("Z_DATA")
```

### Result
```
1
```
