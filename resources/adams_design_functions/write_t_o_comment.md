# WRITE_T_O_COMMENT

Writes a commented line of text to the currently open file. Returns 1 on success and 0 if errors occur.

## Format
```
WRITE_T_O_COMMENT (Comment)
```

## Arguments

**Comment**
: Text to write to file.

## Example

### Function
```
WRITE_T_O_TABLE_LINE ("-------------MDI_HEADER")
```

### Result
```
1
```
