# VAL

Returns an array element nearest the number you specified.

## Format 
```java
VAL(A, x) 
```
## Arguments

**A**
: An array

**x**
: A real number


## Examples 

The following example illustrates the use of the `VAL` function:



### Function  
```java
VAL({2,0,3}, 2.2)
```

### Result  
```java
2
```
