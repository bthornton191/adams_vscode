# PSI

Returns the first angle associated with a body-fixed 313 rotation sequence from one coordinate system object to another. This first rotation is referred to as the psi,  , angle, and is used in association with the theta,  , (2nd rotation) and phi,  , (3rd rotation) angles.

## Format
```
PSI (Object, Reference Frame)
```

## Arguments

**Object**
: Coordinate system object whose rotation is being measured.

**Reference Frame**
: The coordinate system marker with respect to which the rotation is being measured.
