# UNIQUE_NAME

Returns a text string that is a unique database name.

## Format
```java
UNIQUE_NAME (Base Name)
```
## Arguments

**Base Name**
: Starting point for a unique database name.


## Example

The following example illustrates the use of the UNIQUE_NAME function:


### Function  
```java
UNIQUE_NAME("stat")  
```

### Result  
```java
stat_1  
```
