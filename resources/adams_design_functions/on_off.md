# ON_OFF

Returns the character string on or off, depending on the state of the argument.

## Format
```
ON_OFF (State)
```

## Arguments

**State**
: Integer value denoting a Boolean value of on (1) or off (0).

## Example

The following example illustrates the use of the ON_OFF function:

### Function
```
ON_OFF(1)
```

### Result
```
on
```
