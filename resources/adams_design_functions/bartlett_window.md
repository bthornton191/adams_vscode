# BARTLETT_WINDOW

Generate the Bartlett window. 

## Format
```adams_cmd
BARTLETT_WINDOW (n)
```
## Arguments

 



**n** 
: An integer value.  


## Example

The following is an illustration of the BARTLETT_WINDOW function:

 



### Function  
```adams_cmd
bartlett_window (5)  
```

### Result  
```adams_cmd
{0, 0.5000, 1.0000, 0.5000, 0}  
```
