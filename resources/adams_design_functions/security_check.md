# SECURITY_CHECK

Returns a 1 or 0, depending on whether or not the product name is properly licensed.

## Format
```
SECURITY_CHECK (ProductName)
```

## Arguments

**ProductName**
: A character string representing the product name.The string is case sensitive. For example, the product name string for Adams View is AVIEW, so strings such as Aview, aview, AView, will fail.
