# STR_IS_SPACE

Returns 1 (true) if a string is empty; otherwise, returns 0 (false).

## Format
```
STR_IS_SPACE (Input String)
```

## Arguments

**Input String**
: Text string.

## Example

The following examples illustrate the use of the STR_IS_SPACE function:

### Function
```
STR_IS_SPACE("     ")
```

### Result
```
1
```
