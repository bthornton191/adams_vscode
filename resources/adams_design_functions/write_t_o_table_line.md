# WRITE_T_O_TABLE_LINE

Writes a line of text to the currently open file. Returns 1 on success and 0 if errors occur.

## Format
```
WRITE_T_O_TABLE_LINE (Line)
```

## Arguments

**Line**
: Text to write to file.

## Example

### Function
```
WRITE_T_O_TABLE_LINE("$---------STEADY_STATE")
```

### Result
```
1
```
