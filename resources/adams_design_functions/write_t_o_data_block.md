# WRITE_T_O_DATA_BLOCK

Write a block to the currently open file. Returns 1 on success and 0 if errors occur.

## Format
```
WRITE_T_O_DATA_BLOCK (Block Name)
```

## Arguments

**Block Name**
: Block to write to currently open file.

## Example

### Function
```
WRITE_T_O_DATA_BLOCK ("CURVE")
```

### Result
```
1
```
