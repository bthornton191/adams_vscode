# BLACKMAN

Apply the Blackman window to the input array and return the new array. 

## Format
```java
BLACKMAN (a)
```
## Arguments

 



**a** 
: An array 


## Example 

The following is an illustration of the `BLACKMAN` function:

 



### Function  
```java
blackman ({1,2 3,4,2})  
```

### Result  
```java
{0.0000, 0.6800, 3.0000, 1.3600, 0.0000}  
```
