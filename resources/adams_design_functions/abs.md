# ABS

Returns the absolute value of an expression that represents a numerical value. 

## Format 
```adams_cmd
ABS(x) 
```
## Argument 

 



**x** 
: Any valid expression that evaluates to a real number.  


## Example 

The following example illustrates the use of the ABS function:

 



### Function  
```adams_cmd
ABS(3*(-.89))  
```

### Result  
```adams_cmd
2.67  
```
