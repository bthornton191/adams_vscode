# DOT

Returns the dot product of two matrixes.


## Format
```adams_cmd
DOT (M1, M2)

```
## Arguments

 



**M1**
: First matrix.


**M2**
: Second matrix.


## Example

The following example illustrates the use of the `DOT` function:

 

### Function  
```adams_cmd
DOT({1,1,0},{1,0,1})
```

### Result  
```adams_cmd
1
```
